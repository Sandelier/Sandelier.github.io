- This is a website showcasing my (Sandelier's) portfolio.

- Open the terminal on your desktop to begin.

- Use "li" to list directories.

- Navigate with "cd 'directory'" and go back with "cd ..".

- Type "help" for a list of commands and some other commands.

- Enter "read" in the last directory to visit the website.

- Example: "cd extensions/mangapresence" then "read".

- Then to navigate to ext-run after mangapresence: "cd ../../Librarys/ext-run".

- For more pages, use "li" to see available options and navigate through cd.
